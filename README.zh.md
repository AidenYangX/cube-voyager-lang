# cube-voyager-lang

本扩展旨在为 Cube Voyager 的脚本文件在 VSCODE 中提供语言支持。

## 开发中

## Yaml 转换 Json

使用如下语句将 yaml 转换为 json：

```shell
npx js-yaml syntaxes/cvoyager.tmLanguage.yaml > syntaxes/cvoyager.tmLanguage.json
```
